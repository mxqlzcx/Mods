# 更新日志

## [2.0.0] - 2026-01-25（简介.txt定版）

### 重大更新

#### 新增模块
- **TianceAura.lua**（天策军威）：天策上将周围2格内每存在1个友军单位，自身防御力+5，无叠加上限。摒弃无脑无敌设定，以亲兵护卫换取生存能力。
- **TianKeHan.lua**（天可汗系统）：整合万国来朝、节度使、天子一怒三大机制。

#### 新增单位：李建成
- 玄武门建成瞬间在玄武门地块刷新李建成实体
- 李建成：0移动力，超高防御，普通单位攻击强制0伤害
- 仅天策上将的秦王弓可完成绝杀，触发登基/复辟完成

### 数值调整（同步自简介.txt定版）

| 内容 | 旧版 | 定版 |
|------|------|------|
| 贞观之治 | 每城+3%，上限30% | **每城+1%，上限+10%** |
| 万国来朝使者间隔 | 5回合 | **10回合** |
| 藩王线时限 | 35回合 | **30回合** |
| 藩王线惩罚 | -45% | **-50%** |
| 藩王线额外征服要求 | 未定 | **同步完成1城** |

### 架构调整

- 重构 ZhenguanTaizong.lua：贞观之治改为纯 Lua 实现（避免 XML Modifier 加载失败）
- 分离 TianceAura.lua 和 TianKeHan.lua，职责更清晰
- 更新 version.lua，引入 BALANCE_VERSION = "v2.0" 版本标识
- 更新 Main.lua 协调层，引入 TianceGeneralIsAlive() 辅助函数
