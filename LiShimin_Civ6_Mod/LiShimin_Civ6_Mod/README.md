# 天策上将·唐太宗（李世民）

双形态领袖模组，体验从马上打天下到下马治天下的完整历程。

## 模组简介

本模组为《文明6》引入李世民这一独特领袖，设计了革命性的**双形态转换机制**：

1. **天策上将形态**（远古～古典时代）——马上打天下
2. **唐太宗形态**（中世纪及以后）——下马治天下

关键特色：
- 🔄 **动态形态转换**：通过"玄武门之变"事件完成领袖升级
- ⚔️ **特殊单位**：天策上将（远程弓兵）、玄甲军（强力骑兵）
- 📜 **历史事件链**：登基仪式、藩王线、复辟任务
- ⚖️ **版本化平衡**：支持多版本数值配置，便于更新维护

## 快速开始

1. 下载 `.mod` 文件放入《文明6》模组目录
2. 启用模组并开始新游戏
3. 选择领袖"李世民"开始游戏
4. 在古典时代末期进入中世纪时触发关键事件

## 开发指南

### 环境要求

- 《文明6》游戏本体
- ModBuddy（模组开发工具，可从Steam免费获取）
- Python 3.x（用于XML验证工具）

### 编译步骤

1. 用 ModBuddy 打开 `modbuddy/LiShimin.civ6proj`
2. 点击 Build → Compile Mod
3. 生成的文件在 `release/` 目录

### 调试模式

在 `src/Lua/Utils.lua` 中将 `DEBUG_MODE` 设为 `true` 可开启控制台日志输出。
